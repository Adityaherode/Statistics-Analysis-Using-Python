📊 Statistics Analysis Using Python

📌 About:
A Python-based statistical analysis project that applies descriptive statistics, visualization, confidence intervals and hypothesis testing to extract meaningful insights from the Petrol Consumption dataset.

🎯 Objectives
* Clean and preprocess the dataset.
* Perform descriptive statistical analysis.
* Visualize data distribution and identify outliers.
* Calculate Confidence Intervals (90%, 95%, and 99%).
* Perform Hypothesis Testing.
* Interpret statistical results and generate insights.

🛠️ Technologies Used
* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SciPy

📊 Dataset Analysis:
The Petrol Consumption dataset was cleaned and analyzed using descriptive statistics and data visualization techniques. Histograms, KDE plots, box plots, scatter plots and correlation heatmaps were used to understand data distribution and relationships between variables.

Outlier detection using the IQR method identified extreme values in Petrol Tax, Paved Highways, Driver Licence Population and Petrol Consumption. After removing these outliers, the dataset became more consistent and suitable for statistical analysis.

The analysis showed a positive relationship between Driver Licence Population and Petrol Consumption, indicating that regions with more licensed drivers generally consume more fuel. Paved Highways also positively influenced petrol consumption, while Petrol Tax showed a negative relationship, suggesting that higher fuel taxes tend to reduce petrol usage.

Confidence Interval analysis was performed at 90%, 95% and 99% confidence levels. The results showed that higher confidence levels produced wider confidence intervals, providing greater certainty about the population mean while reducing precision.

Hypothesis Testing was conducted at a 5% significance level (α = 0.05). In both statistical tests, the Null Hypothesis (H₀) was accepted, indicating that there was no statistically significant difference between the sample evidence and the assumed population mean.

📈 Key Findings:
* Descriptive statistics summarized the central tendency and variability of the dataset.
* Outliers were successfully detected and removed using the IQR method.
* Driver Licence Population has a positive impact on Petrol Consumption.
* Petrol Tax has a negative impact on Petrol Consumption.
* Better road infrastructure is associated with higher fuel consumption.
* Higher confidence levels result in wider confidence intervals.
* Both hypothesis tests supported the Null Hypothesis (H₀).

✅ Conclusion:
This project demonstrates the practical application of statistical techniques using Python. It combines descriptive statistics, visualization, outlier detection, confidence intervals and hypothesis testing to generate reliable insights and support data-driven decision-making.
